Introduction¶

In this assignment, unleaded gasoline sales(UGS) for each quarter in 2007 are aimed to be forecasted by using the time series approaches and some potentially related input variables mentioned belove:

UGS: Unleaded gasoline sale in a given quarter,
RNUV: An index indicating the rate of new unleaded gasoline using vehicles being added to the traffic in a quarter,
PU: Average price (adjusted with an index) of a liter of unleaded gasoline in a quarter,
PG: Average price (adjusted with an index) of a liter of diesel gasoline in a quarter,
NUGV: Number of unleaded gasoline using vehicles in the traffic,
NDGV: Number of diesel gasoline using vehicles in the traffic (per 1000 people),
GNPA: Agriculture component of Gross National Product (adjusted with an index),
GNPC: Commerce component of Gross National Product (adjusted with an index),
GNP: Grand total for GNP (agriculture, commerce and other components total). </font>

4.2 Selecting the Best Model¶

After constructing different models and getting some insights about the effects of the variables, I constructed the model above as by best model. Although a better regression model can be constructed, I decided to move forward with this model by taking initiative due to the lack of data. For instance, the trend and PU variables are kept in the model even though the above models generally interpret them as insignificant input variables. At the end of several trials, I have achieved a model, in which all input variables are significant.
The adjusted R^2 value of my best model is : 0.9679

4.2.1 Statistical Properties of the Selected Model¶

Residual Analysis

When the distribution of the residuals is investigated, the variance of the residuals does not have constant variance and therefore, the normalization assumption of residuals is violated at some degree. This can be seen from both interpreting the graph of the residuals, and plotting the Q-Q plot. Moreover, there is a high correlation between the residuals at lag 4, but when the Y_t_4 variable is added to the model, the model performs worse, therefore, no action was taken. Although the selected model has some unwanted properties, it is selected since it is the best option when it is compared to the other models tried before.
